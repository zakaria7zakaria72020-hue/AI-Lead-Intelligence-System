# 🤖 AI-Lead Intelligence System

### 🎯 Overview
This project is a back-end engine that automates lead qualification. It receives raw data via Webhooks, processes it through custom business logic, and outputs actionable insights for sales teams.

### 🛠 Tech Stack
- **Backend:** Python (Flask)
- **Deployment:** Cloud-ready (PythonAnywhere)
- **Integration:** Designed for n8n & Slack workflows

### 🚀 How it works
1. **Receive:** Captures lead data (Name, Budget, Interest).
2. **Analyze:** Runs a logic-check to prioritize leads.
3. **Notify:** Sends a structured JSON response to automation platforms.
